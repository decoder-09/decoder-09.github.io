This is a small multiple choice quiz game. The user will be presented with a welcome screen and instructions on how to play the game. User is presented with a series of questions with 4 multiple choice options. User will select/click an answer choice and the next question and answer set will be presented. When all questions have been answered, or the timer has run out, user score is calculated and user can enter their name and be added to the list of persisted scores.
